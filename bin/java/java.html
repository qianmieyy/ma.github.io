JAVA有哪几种访问修饰符？有哪些限制
访问权限：（从小到大）
private:只有自己能访问
package：包内都能访问（什么都不写就是包级别）
protected：受保护的，可以跨包，但要有继承关系。（家族的）
public：公共的，都可以访问

重写：把父类的方法覆盖一遍。注意：访问修饰符只能大于等于父类方法

调用父类方法用什么关键字？super.方法

构造函数（构造方法）：方法名和类同名，无返回类型，用new关键字来调用.
                                    作用是：初始化对象
注意：无参构造，系统会默认提供一个。
          如果你写了构造函数，系统则会把默认提供的收回。
          先调用父类，再调用子类
          子类构造中，如何调用父类有参构造？super(参数);
注意：super只能放在第一行第一时间调用
